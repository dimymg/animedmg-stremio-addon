# AnimeDmg+ Stremio Addon (PT-BR)

Addon gratuito que entrega episódios dublados ou legendados em português.

## Deploy 1-clique no Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/neysec/animefire-stremio-addon)

1. Clique no botão acima.  
2. Deixe o **Build Command** como `npm install` e o **Start Command** como `node index.js`.  
3. Copie a URL gerada (`https://seuapp.onrender.com`).

## Instalar no Stremio

- Abra Stremio → **Addons** → **Install from URL**  
- Cole: `https://seuapp.onrender.com/manifest.json`

## Testar

Navegador:  
`https://seuapp.onrender.com/stream/series/animefire:naruto-classico-ep-001.json`
